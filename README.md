# Despertador Web

Um despertador simples e elegante para uso no trabalho, desenvolvido em HTML, CSS e JavaScript puro.

## Funcionalidades

- Relógio em tempo real no horário de Brasília
- Adicionar, editar e remover alarmes
- Até 10 alarmes simultâneos
- Tema claro e escuro
- Persistência dos alarmes no navegador
- Modal de alarme com som
- Interface responsiva

## Como Usar

1. Abra o arquivo `index.html` em um navegador web.
2. Defina o horário do alarme usando o campo de entrada.
3. Clique no botão "+" para adicionar o alarme.
4. Quando o horário chegar, o alarme tocará e um modal aparecerá.
5. Clique em "PARAR ALARME" para parar.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Web Workers para atualização do relógio

## Instalação

1. Clone o repositório:
   ```
   git clone https://github.com/Davemottaa/Despertador.git
   ```
2. Abra o arquivo `index.html` em um navegador web moderno.
